# 家政管理系统 java1072

## 项目概述

家政管理系统是一款集员工管理、订单处理、客户服务于一体的高效、可靠的信息化解决方案。该系统采用前后端分离的技术架构，前端基于Vue.js框架，后端采用Python的Django框架。

## 技术栈

- **前端**: Vue.js
- **后端**: Python 3, Django
- **数据库**: PostgreSQL
- **其他**: Redis, Celery

## 功能模块

- **用户管理**: 系统角色分配，权限控制
  - 用户认证
  - 角色授权
- **员工管理**: 员工信息维护，技能认证
  - 员工信息录入
  - 技能管理
- **订单管理**: 客户订单处理，状态跟踪
  - 订单创建
  - 订单状态更新
- **客户服务**: 客户资料管理，服务反馈
  - 客户资料维护
  - 服务评价

## 系统角色

- 系统管理员
- 客户服务人员
- 家政服务人员

## 运行环境

- **操作系统**: Linux或Unix-like
- **Python版本**: Python 3.8 或更高版本
- **数据库**: PostgreSQL 12 或更高版本
- **其他依赖**: Redis, Node.js

## 部署步骤

1. 环境准备
   - 安装Python 3.8 或更高版本
   - 安装PostgreSQL 12 或更高版本
   - 安装Redis
   - 安装Node.js
2. 源代码部署
   - 后端代码部署
   - 前端代码部署
3. 数据库迁移与初始化
4. 应用配置
5. 启动服务

## 核心亮点

- **前后端分离**: 提高开发效率，降低维护成本
- **权限控制**: 灵活的用户和角色管理，保障系统安全
- **高效可靠**: Python和Django保证了系统的快速响应和稳定性
- **易于扩展**: 设计上考虑了未来功能的拓展性，便于添加新模块或功能
- **自动化工作流**: 利用Celery和Redis处理后台任务，提高系统效率
